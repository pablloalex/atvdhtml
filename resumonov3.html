Resumo - Capítulo 3 CSS

1. O que é CSS

CSS significa Cascading Style Sheets. É a linguagem usada para estilizar páginas HTML.

Com CSS podemos mudar:

  • Cores
  • Fontes
  • Tamanhos
  • Espaçamentos
  • Layouts
h1 {
  color: blue;
  font-size: 30px;
}

2. Formas de usar CSS

CSS Externo

Usa um arquivo separado .css.

<link rel="stylesheet" href="style.css">

CSS Interno

Usa a tag <style> dentro do HTML.

CSS Inline

Usa o atributo style diretamente no elemento.

<p style="color:red;">Texto</p>

3. Seletores

Seletor Exemplo
Tag p { }
Classe .menu { }
ID #topo { }

Os seletores servem para escolher quais elementos serão estilizados.

4. Cores e Textos

Existem várias formas de usar cores:

  • red
  • #ff0000
  • rgb(255,0,0)

Principais propriedades:

  • color → cor do texto
  • background → fundo
  • font-size → tamanho da fonte
  • text-align → alinhamento

5. Layout

CSS também é usado para organizar os elementos da página.

Flexbox

.container {
  display: flex;
}

Grid

.container {
  display: grid;
}

Flexbox é usado para linhas e colunas simples. Grid é usado para layouts mais completos.

6. Pontos Importantes

  • CSS estiliza páginas HTML
  • Pode ser externo, interno ou inline
  • Usa seletores para escolher elementos
  • Controla cores, textos e layouts
  • Flexbox e Grid ajudam na organização
