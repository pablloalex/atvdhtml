Resumocap2 - Estruturação de Páginas Web com HTML

HTML (HyperText Markup Language) é a linguagem usada para criar e estruturar páginas da web. Ela define títulos, parágrafos, imagens, links, listas, tabelas e formulários.

Estrutura Básica do HTML

Uma página HTML possui elementos principais como:

  • <html> → inicia a página
  • <head> → informações da página
  • <title> → título da aba
  • <body> → conteúdo visível

Elementos e Atributos

Os elementos HTML são formados por tags de abertura e fechamento. Os atributos adicionam informações extras aos elementos.

Exemplo: 

<a href="https://google.com">Google</a>

Textos e Formatação

O HTML possui elementos para organizar textos:

  • <h1> até <h6> → títulos
  • <p> → parágrafos
  • <b> → texto em negrito
  • <i> → texto em itálico
  • <br> → quebra de linha

Listas

Existem três tipos principais de listas:

  1. Listas ordenadas (<ol>)
  2. Listas não ordenadas (<ul>)
  3. Listas de definição (<dl>)

Links

Os links são criados com a tag <a> e permitem acessar outras páginas ou partes da mesma página.

Imagens

Imagens são adicionadas usando a tag <img>. O atributo "src" define o caminho da imagem e o "alt" adiciona uma descrição.

Tabelas

As tabelas organizam dados em linhas e colunas.

  • <table> → tabela
  • <tr> → linha
  • <td> → célula
  • <th> → cabeçalho

Formulários

Formulários permitem que usuários enviem informações.

  • Campos de texto
  • Senha
  • Botões
  • Checkbox
  • Listas de seleção

Conclusão

HTML é a base da criação de páginas web. Com ele é possível estruturar conteúdos de forma organizada e preparar páginas para receber estilos com CSS e funcionalidades com JavaScript.
